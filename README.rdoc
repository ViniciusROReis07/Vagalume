
# Consumo da API disponibilizada por Vagalume.




Este projeto da aula de PWFE do SENAI teve como objetivo fixar os aprendizados em relação ao consumo de APIs.

1-Competências.

1.2-Consumo de uma API a partir de sua documentação e links disponibilizados. 

1.3-Realização uma página simples, porém dinâmica.

1.4-Documentação simples de objetivos e competências.

1.5-Pesquisa, independente, em diversos meios para o saneamento de necessidades que surgem conforme ocorre o progresso da ferramenta.

1.6-Localização e correção de bugs.

1.7-Reutilização de componentes feitos anteriormente.

1.8-Administração de tempo de confecção de projeto.


1.9-Manipular elemento de HTML e CSS por meio do Javascript



2-Possíveis erros e suas soluções

2.1-Em caso de qualquer erro entre em contato com o desenvolvedor



3-Passo a passo do consumo da API

3.2-Primeiro é necessário ir à página do fornecedor da API na área de documentação: http://api.vagalume.com.br/docs/ .


3.3-Essa pagina ira mostrar dados sobre sua api e terá urls de acordo com a resposta que você deseja.

3.4-Url utilizada neste projeto foi : https://www.vagalume.com.br/{neste campo ira o nome do artista}/index.js;
